💳 Enterprise-Grade Financial Fraud Detection System

An end-to-end machine learning system for detecting fraudulent financial transactions using advanced ML, anomaly detection, explainable AI, and production-ready deployment.

---

🚀 Project Overview

Financial institutions lose millions due to fraudulent transactions. This project builds a production-style fraud detection system that:

- Handles extreme class imbalance
- Compares multiple ML models
- Implements anomaly detection
- Uses explainable AI (SHAP)
- Optimizes fraud cost impact
- Provides a real-time API backend
- Is fully Dockerized for deployment

---

🧠 Models Implemented

- Logistic Regression (Baseline)
- LightGBM (Gradient Boosting)
- Isolation Forest (Unsupervised Anomaly Detection)
- Deep Autoencoder (PyTorch)

---

📊 Evaluation Strategy

Since fraud datasets are highly imbalanced, the system uses:

- Precision-Recall Curve
- ROC-AUC
- Confusion Matrix
- Cost-based Fraud Loss Simulation

Business-aware evaluation ensures financial loss minimization rather than just accuracy optimization.

---

🔍 Explainable AI

Implemented SHAP (SHapley Additive Explanations) to:

- Identify key fraud-driving features
- Provide global and local interpretability
- Improve trust in predictions
